# lookart-ip
Powerfull python api to lookup IP's.

It can provide:
```
-city
-region
-region code
-country
-country name
-continent code
-in europe?
-postal
-latitude
-longitude
-timezone
-utc offset
-country calling code
-currency
-languages
-asn
-org/ISP
```

## setup:
```
pip install requests
```

## Built with
* [Python 3](https://www.python.org/downloads/) - The language used.
* Python modules used: requests, json.
